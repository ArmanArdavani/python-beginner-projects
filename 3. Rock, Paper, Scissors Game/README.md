# Rock, Paper, Scissors 🪨📃✂️

A command-line Rock, Paper, Scissors game built in Python.

## Features

- Play against the computer or a second player
- First to win 2 out of 3 rounds wins the game
- Win/loss/tie statistics displayed at the end
- Input validation on all prompts
- Emoji display for choices

## How to Run

Make sure you have Python 3 installed, then:

```bash
python main.py
```

## How to Play

1. Choose your game mode: `two player` or `computer`
2. Each round, enter your choice:
   - `r` — Rock 🪨
   - `p` — Paper 📃
   - `s` — Scissors ✂️
3. First player to win 2 rounds wins the match
4. After each game, choose to play again or quit
